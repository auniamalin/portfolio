SleepWell+

Smart Sleep Tracking & Wellness Improvement System

Mobile Application | Individual Project | AI-Powered Health & Sleep Monitoring

Project Overview

SleepWell+ is a mobile sleep tracking application designed to help users develop healthier sleep habits through manual sleep logging, personalized insights, and structured sleep reports. Unlike sensor-based sleep trackers, SleepWell+ focuses on simple user input while still providing meaningful analytics to improve sleep quality and consistency.

The system allows users to record their sleep and wake-up times daily, where the app automatically calculates total sleep duration. It also introduces a Sleep Habit Score (0%–100%) that evaluates sleep consistency based on duration, pattern regularity, and wake-up stability.

To support habit formation, SleepWell+ generates daily and weekly sleep reports that visualize trends in sleep duration and consistency. All sleep data is securely stored in a database, allowing users to track long-term progress and improve their sleep health over time.

System Features

Sleep Logging

  • Users manually log sleep and wake-up times daily.
  • Ensures simple tracking without external sensors.
  • All entries stored securely in the database.

Sleep Duration Calculation

  • Automatically calculates total sleep duration.
  • Removes manual calculation for users.
  • Helps users monitor sleep goals easily.

Sleep Habit Score

  • Scores sleep consistency from 0% – 100%.
  • Based on duration, pattern consistency, and wake-up stability.
  • Helps users track improvement over time.

Daily & Weekly Reports

  • Displays sleep trends and analytics.
  • Shows average sleep duration and consistency changes.
  • Helps users understand sleeping patterns visually.

Database Storage

  • Stores sleep data securely in Firebase database.
  • Allows retrieval of past sleep records.
  • Supports long-term habit tracking.
